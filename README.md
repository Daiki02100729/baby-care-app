# 医療ケア児ノート
医療ケア児を持つ家族向けの記録アプリです。
バイタル・吸引・胃瘻・投薬・排泄などを記録できます。

## 開発背景
長女は医療ケア児でした。
育児記録アプリの「ぴよろぐ」のようなツールを使いたかったのですが、
医療ケア児に対応したものが見当たらず、同じ境遇の家族に届けたいという想いから、
このアプリの開発を始めました。

## 使用技術
HTML / CSS / JavaScript

## 開発状況
バイタル・吸引・胃瘻・経管栄養・人工呼吸器・投薬・排泄の記録機能を実装済み。
